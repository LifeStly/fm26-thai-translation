# ภาษาไทยสำหรับ Football Manager 2026 (FM26 Thai Language)

ไฟล์แปลภาษาไทยสำหรับเกม **Football Manager 2026**

> ดาวน์โหลดไฟล์ `thai.ltc` ได้ที่หน้า **[Releases](../../releases/latest)** (ปุ่มทางขวาของหน้า repo)

---

## 📥 วิธีติดตั้ง

**1. ดาวน์โหลดไฟล์**
โหลด `thai.ltc` จากหน้า [Releases](../../releases/latest)

**2. เปิดโฟลเดอร์ภาษาของเกม**
กดปุ่ม `Win + R` แล้วพิมพ์ path นี้ลงไป (หรือก็อปวางในแถบที่อยู่ของ File Explorer):

```
%USERPROFILE%\Documents\Sports Interactive\Football Manager 26\languages
```

- ถ้าไม่มีโฟลเดอร์ชื่อ `languages` → **สร้างขึ้นมาเอง** ในโฟลเดอร์ `Football Manager 26`
- ถ้าใช้ OneDrive แล้วหาไม่เจอ ลองดูที่ `%USERPROFILE%\OneDrive\Documents\Sports Interactive\...` แทน

**3. วางไฟล์**
ก็อปไฟล์ `thai.ltc` ที่โหลดมา ลงในโฟลเดอร์ `languages`

**4. เลือกภาษาในเกม**
เปิด FM26 แล้วไปที่:

```
Preferences (ตั้งค่า)  →  Interface  →  Language / Localisation
```

เลือกภาษา **"ไทย"** จากเมนู → กด **Confirm**

**5. ถ้าภาษายังไม่เปลี่ยน**
ปิดแล้วเปิดเกมใหม่อีกครั้ง

---

## ❓ ปัญหาที่เจอบ่อย

| อาการ | วิธีแก้ |
|-------|---------|
| ไม่เห็นภาษา "ไทย" ในเมนู | เช็กว่าวางไฟล์ใน `languages` ถูกที่ และชื่อไฟล์เป็น `thai.ltc` (ไม่ใช่ `thai.ltc.txt`) |
| หาโฟลเดอร์ Documents ไม่เจอ | ลองดูที่ OneDrive\Documents — Windows บางเครื่องย้าย Documents ไปไว้ใน OneDrive |
| บางข้อความยังเป็นอังกฤษ | ปกติ — บางส่วนยังแปลไม่ครบ 100% จะอัปเดตในเวอร์ชันถัดไป |

---

## ℹ️ ข้อมูลไฟล์

- `thai.ltc` — ไฟล์ภาษาที่เกมใช้งานจริง (compiled language file) วางไฟล์เดียวก็เล่นได้เลย
- เป็นงานแปลโดยแฟนเกม (fan translation) — ไม่ใช่ของ Sports Interactive อย่างเป็นทางการ

หากเจอจุดที่แปลผิดหรือแปลตกหล่น เปิด [Issue](../../issues) แจ้งได้เลยครับ 🙏
